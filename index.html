RxJS Observable Playground

Available in Console

Global Variables

Rx

A reference to the RxJS library.

observer

A sample observer which logs events to the console.
Rx.Observable.fromEvent(button,'click').subscribe(observer);

observable

A sample observable which returns 42 then completes.
observable.subscribe(event=>console.log(event));

subject

A plain RxJS subject.
subject.subscribe(e=>console.log(e));subject.next(42);

behaviorSubject

An RxJS behavior subject configured to return 42 as an initial value. New subscribers automatically get the last value it sent.
behaviorSubject.subscribe(observer);
behaviorSubject.next("Observe THIS!");behaviorSubject.subscribe(observer);

replaySubject

An RxJS replay subject that has already sent 2 values. New subscribers automatically get the last 3 values it sent.
replaySubject.subscribe(observer);
replaySubject.next("Stark");replaySubject.next("Bolton"); replaySubject.subscribe(observer);

Static Operators

interval(t)

Shortcut for Rx.Observable.interval. Creates an observable which pushes an event every t milliseconds.
interval(1000).subscribe(observer)
